# 🛡️ AI Surveillance System (YOLOv8 + Tracking + Streamlit)

A real-time AI-powered surveillance system that detects, tracks, and analyzes human movement using YOLOv8 and OpenCV, with a live Streamlit dashboard.

---

## 🚀 Features

- 👤 Real-time person detection (YOLOv8)
- 🔄 Object tracking with unique IDs
- 🔢 Entry/Exit counting (line crossing)
- 🚫 Intrusion detection (restricted zone)
- 📊 Live dashboard using Streamlit

---

## 🧠 Tech Stack

- Python
- OpenCV
- YOLOv8 (Ultralytics)
- Streamlit
- NumPy
